# Tetris CLI 🎮

Классическая игра Тетрис для терминала Linux/Unix, написанная на Rust.

![Скриншот игры](docs/img/screenshot.png)

## 📖 Описание

**Tetris CLI** — это полноценная реализация легендарной игры Тетрис, работающая прямо в вашем терминале. Игра поддерживает все классические механики: вращение фигур, мгновенное падение, подсчёт очков и сохранение рекордов.

### Особенности

- ✨ **Классический геймплей** — все 7 типов тетрамино (T, L, J, S, Z, O, I)
- 🎨 **Цветная графика** — каждая фигура имеет свой цвет
- 🏆 **Система рекордов** — лучший результат сохраняется между запусками
- ⚡ **Плавная анимация** — 60 FPS для комфортной игры
- 🎯 **Прогрессивная сложность** — скорость падения увеличивается с каждым уровнем
- ⏸️ **Пауза** — возможность приостановить игру в любой момент
- 👻 **Призрачная фигура** — показывает, куда упадёт фигура
- 👀 **Предпросмотр** — показ следующей фигуры
- 📊 **Таблица лидеров** — топ-5 лучших результатов

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `a` | Движение влево |
| `d` | Движение вправо |
| `q` | Поворот фигуры против часовой стрелки |
| `e` | Поворот фигуры по часовой стрелке |
| `s` | Мгновенное падение (hard drop) |
| `p` | Пауза / Продолжить |
| `Backspace` | Выход в меню / Выход из паузы |
| `Enter` | Начать игру (в меню) |
| `l` | Показать таблицу лидеров |

## 📋 Системные требования

- **ОС**: Linux или другая Unix-подобная система
- **Терминал**: Поддерживающий raw-режим и ANSI-цвета
- **Минимальный размер окна**: 22×25 символов

> ⚠️ **Важно**: Игра не работает в Windows Terminal. Для запуска в Windows используйте WSL (Windows Subsystem for Linux).

## 🎯 Правила игры

### Начисление очков

- **За фигуру**: 100 + (скорость падения × 50) очков
- **За 1 линию**: 100 очков
- **За 2 линии**: 200 очков
- **За 3 линии**: 400 очков
- **За 4 линии**: 800 очков
- **Бонус за уровень**: 500 × (номер уровня - 1)

### Прогрессия сложности

- За каждую заполненную линию скорость падения увеличивается на 0.05
- Начальная скорость: 0.9 блоков/секунду
- Уровень повышается каждые 10 удалённых линий

### Проигрыш

Игра заканчивается, когда новая фигура не может появиться на поле (заблокирована другими фигурами).

## 🧪 Тестирование

Проект содержит 20 модульных тестов, покрывающих:

1. **Tetromino (4 теста)**: создание фигур, вращение, координаты

4. **Leaderboard (4 теста)**: добавление, сортировка, валидация
5. **Константы и границы (2 теста)**: размеры поля, проверка границ

## Руководство пользователя

### Быстрый старт

1. Запустите игру: `cargo run`
2. В меню нажмите **Enter** для начала
3. Используйте **a/d** для перемещения, **q/e** для вращения
4. Нажмите **s** для мгновенного падения
5. Нажмите **p** для паузы

### Советы для новичков

1. **Держите поле плоским** — избегайте высоких столбцов
2. **Оставляйте колодец** — оставьте одну колонку пустой для I-фигуры
3. **Планируйте наперёд** — смотрите на следующую фигуру
4. **Используйте призрачную фигуру** — она показывает точку приземления

### Продвинутые техники

#### T-Spin
Вращение T-фигуры в последнюю минуту для размещения в узких местах.

#### Комбо
Постоянное удаление линий создаёт цепную реакцию очков.

#### Удержание центра
Старайтесь держать самые высокие блоки в центре поля.

### Решение проблем

#### Игра не запускается
**Проблема**: "Окно терминала слишком маленькое"
**Решение**: Увеличьте окно до минимального размера 22×25 символов

#### Игра не отображает цвета
**Решение**: Убедитесь, что терминал поддерживает ANSI-цвета

#### Задержка ввода
**Решение**: Закройте другие программы, убедитесь в стабильных 60 FPS
