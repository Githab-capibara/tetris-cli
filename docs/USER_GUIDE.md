# Руководство пользователя Tetris CLI

## Быстрый старт

### Запуск игры

```bash
cargo run
```

Или, если игра уже установлена:

```bash
tetris-cli
```

### Минимальные требования

- **ОС**: Linux или Unix-подобная система
- **Размер терминала**: минимум 22×25 символов
- **Поддержка**: ANSI-цвета и raw-режим терминала

> ⚠️ **Внимание**: Игра не работает в Windows Terminal. Используйте WSL (Windows Subsystem for Linux) для запуска в Windows.

## Начало игры

1. Запустите игру командой `cargo run`
2. В главном меню нажмите **Enter** для начала игры
3. Нажмите **Backspace** для выхода из игры

## Управление

### Основные команды

| Клавиша | Действие | Иллюстрация |
|---------|----------|-------------|
| **a** | Переместить фигуру влево | ← |
| **d** | Переместить фигуру вправо | → |
| **q** | Повернуть против часовой стрелки | ↶ |
| **e** | Повернуть по часовой стрелке | ↷ |
| **s** | Мгновенное падение (hard drop) | ↓↓ |
| **p** | Пауза / Продолжить | ⏸️ |
| **Backspace** | Выход в меню / Выход из паузы | ⎋ |

### Советы по управлению

- **Мгновенное падение (s)**: Фигура мгновенно опускается на дно и фиксируется. Используйте для ускорения игры.
- **Вращение (q/e)**: Некоторые фигуры (квадрат O) не вращаются.
- **Пауза (p)**: Во время паузы можно выйти из игры нажатием Backspace.

## Игровой процесс

### Цель игры

Размещайте падающие фигуры (тетрамино) так, чтобы заполнять горизонтальные линии без промежутков. Заполненные линии исчезают, а вы получаете очки.

### Типы фигур

```
T-образная    L-образная    J-образная
  █             █             █
███            █            ███
               ██             █

S-образная    Z-образная    Квадрат      Линия
 ██            ██                           █
██            ██                           ███
                             ██             █
```

### Система очков

#### Начисление очков за фигуры

| Действие | Очки |
|----------|------|
| Падение фигуры | 100 + (скорость × 50) |
| 1 заполненная линия | 100 |
| 2 заполненные линии | 200 |
| 3 заполненные линии | 400 |
| 4 заполненные линии | 800 |

#### Бонус за комбо

При одновременном удалении нескольких линий очки умножаются:
- 2 линии: ×2
- 3 линии: ×4
- 4 линии: ×8

### Прогрессия сложности

| Уровень | Скорость | Линий для перехода |
|---------|----------|-------------------|
| 1 | 0.90 | 0 |
| 2 | 0.95 | 1 |
| 3 | 1.00 | 2 |
| 4 | 1.05 | 3 |
| 5 | 1.10 | 4 |
| ... | ... | ... |

**Примечание**: Скорость увеличивается на 0.05 за каждую заполненную линию.

### Окончание игры

Игра заканчивается, когда новая фигура не может появиться на поле (верхние клетки заблокированы).

```
╔════════════╗
║ ИГРА ОКОНЧЕНА ║
╚════════════╝
```

После этого показывается ваш финальный счёт.

## Рекорды

### Как работают рекорды

- Лучший результат автоматически сохраняется между запусками
- Рекорд защищён от подделки криптографическим хешем
- При обнаружении подделки рекорд сбрасывается в 0

### Просмотр рекорда

Рекорд отображается:
- В главном меню
- Во время игры в верхней части экрана

### Сброс рекорда

Рекорд хранится в конфигурационном файле. Для сброса удалите файл конфигурации:

```bash
# Путь к конфигу зависит от ОС
# Linux: ~/.config/tetris-cli/config.toml
# macOS: ~/Library/Preferences/tetris-cli/config.toml
```

## Стратегии игры

### Советы для новичков

1. **Держите поле плоским** — избегайте высоких столбцов
2. **Оставляйте колодец** — оставьте одну колонку пустой для фигуры-линии (I)
3. **Планируйте наперёд** — смотрите, куда упадёт следующая фигура
4. **Используйте вращение** — некоторые позиции доступны только через вращение

### Продвинутые техники

#### T-Spin
Вращение T-фигуры в последнюю минуту для размещения в узких местах.

#### Комбо
Постоянное удаление линий создаёт цепную реакцию очков.

#### Удержание центра
Старайтесь держать самые высокие блоки в центре поля для большей гибкости.

## Решение проблем

### Игра не запускается

**Проблема**: "Невозможно запустить игру! Окно терминала слишком маленькое"

**Решение**:
1. Увеличьте окно терминала
2. Минимальный размер: 22×25 символов
3. Уменьшите масштаб шрифта терминала

### Игра не отображает цвета

**Проблема**: Фигуры отображаются без цветов

**Решение**:
1. Убедитесь, что терминал поддерживает ANSI-цвета
2. Попробуйте другой терминал (gnome-terminal, konsole, xterm)

### Задержка ввода

**Проблема**: Фигуры реагируют на нажатия с задержкой

**Решение**:
1. Закройте другие программы, использующие процессор
2. Убедитесь, что FPS не проседает (60 FPS требуется для плавной игры)

### Игра не работает в Windows Terminal

**Проблема**: При запуске в Windows возникает ошибка

**Решение**:
1. Установите WSL (Windows Subsystem for Linux)
2. Запустите игру внутри WSL-терминала
3. Альтернативно: используйте Linux-подобную ОС

## Часто задаваемые вопросы (FAQ)

### Можно ли изменить управление?

В текущей версии управление фиксировано. Изменение клавиш возможно только через модификацию исходного кода.

### Как сохранить рекорд?

Рекорд сохраняется автоматически при завершении игры, если вы превзошли предыдущий.

### Есть ли звуковые эффекты?

Нет, игра работает без звука. Только визуальный интерфейс в терминале.

### Можно ли играть мышью?

Нет, управление только с клавиатуры.

### Сколько максимум очков можно набрать?

Теоретического лимита нет, но на практике игра становится очень быстрой после 50-100 линий.

### Что происходит, когда скорость становится очень большой?

Фигуры падают почти мгновенно, что делает игру крайне сложной.
